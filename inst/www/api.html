API


There are three ways to use DateLife. One is to use the datelife.org website. Another is to use the datelife R package. The third is to use HTTP POST and GET requests. This page documents that.
The API is based on that of of the OpenCPU project: see this PDF for more info.
[Change URL when deployed live] 
curl http://localhost:4361/ocpu/library/datelifeweb/R/GetAllCitations/ -d ""
returns 
/ocpu/tmp/x0cf1b948fb/R/.val
/ocpu/tmp/x0cf1b948fb/stdout
/ocpu/tmp/x0cf1b948fb/warnings
/ocpu/tmp/x0cf1b948fb/source
/ocpu/tmp/x0cf1b948fb/console
/ocpu/tmp/x0cf1b948fb/info
/ocpu/tmp/x0cf1b948fb/files/DESCRIPTION

Which are addresses for later GET requests: 
curl http://localhost:4361/ocpu/tmp/x0cf1b948fb/R/.val/print
Reuse encouraged
